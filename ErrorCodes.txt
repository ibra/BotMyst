These are the error codes for BotMyst. Whenever catching an error, the appropriate code should be used.
 
Error 400:  Bad/Missing Arguments - Self-Explanatory
Error 403:  Missing Permissions / Denied Access - Self-Explanatory
Error 404:  Not Found - Self-Explanatory
Error 424:  Something failed because it depended on another request and that request failed.
Error 429:  Argument Limit Reached - When a given argument has a value greater than one it requires.
